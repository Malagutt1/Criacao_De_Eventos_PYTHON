Explicações sobre tags, propriedades, abordagens usadas na paǵina "links.html"

Este exemplo ilustra como estruturar links internos e externos em um documento HTML, facilitando a navegação dentro e fora do site. A habilidade de interligar conteúdos é fundamental para a construção de uma boa experiência do usuário na web.

A tag <a> é usada para criar hyperlinks. Ela tem vários atributos, mas os mais importantes são href e target.

href: Especifica o destino do link. Pode ser um URL para uma página externa, um caminho para uma página interna ou um identificador de seção precedido por # para navegação interna na mesma página.

target: Especifica onde abrir o link. O valor _blank abre o link em uma nova aba do navegador. Este atributo é comumente usado para links externos para manter os visitantes no site original enquanto exploram o link externo.

rel="noopener noreferrer": Este atributo é uma prática de segurança recomendada para links que usam target="_blank", protegendo contra vulnerabilidades e passagem de informações de referência.

Links Internos Link para uma seção na mesma página: Usando href="#secao-informacoes", o link pula para a parte da página com o id "secao-informacoes".

Link para outra página no mesmo site: href="pagina-local.html" aponta para um arquivo no mesmo diretório ou caminho especificado.

Links Externos Link para um site externo: href="https://www.google.com" define o URL do site de destino, e target="_blank" faz com que o link seja aberto em uma nova aba.


Códigos HTML para caracteres especiais

Os códigos HTML para caracteres especiais começam com "&#" e terminam com ";". Por exemplo, o sinal de percentual (%) tem um código numérico de "#&37;" e um código hexadecimal de "#&x25;".
