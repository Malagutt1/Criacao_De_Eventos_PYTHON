Notas sobre Áudio e Vídeo

Esta página explica como usar as tags <audio> e <video> para incorporar mídia em páginas web.

Funcionalidades disponíveis:

  • O atributo controls adiciona botões para tocar, pausar, ajustar volume, entre outros.
  • Atributos como autoplay, loop e muted regulam o comportamento da mídia.
  • O navegador seleciona o primeiro <source> compatível com seu formato suportado.

Restrições e cuidados:

  • Nem todos os navegadores suportam todos os tipos de arquivo.
  • Áudio: Formato MP3 tem maior compatibilidade.
  • Vídeo: Formato MP4 é o mais aceito.
  • Os atributos width e height aplicam-se somente a vídeos, não a áudios.
  • O conteúdo dentro das tags <audio> ou <video> aparece caso o navegador não suporte a mídia.

O que pode não funcionar:

  • Versões antigas de navegadores que não suportam HTML5.
  • Autoplay com som pode ser bloqueado em dispositivos móveis.

Recomendações:

  • Evite usar autoplay com áudio ativo para melhorar a experiência do usuário.
  • Inclua sempre o atributo controls para que o usuário possa controlar a reprodução.
  • Otimize o tamanho dos arquivos de mídia para acelerar o carregamento.
  • Utilize loop com cautela, somente quando a repetição for necessária.
