Explicação: Tags e Propriedades

Algumas tags que devem ser usadas na sua página HTML são:

  • <h1>...</h1> — Título principal da página
  • <p>...</p> — Parágrafo de texto
  • <ul>...</ul> — Lista não ordenada
  • <li>...</li> — Item da lista
  • <a>...</a> — Link para outra página ou site
  • <img> — Imagem na página

Diferentes tipos de lista para organizar uma página web

  • <ol> (Ordered List): representa uma lista ordenada que automaticamente enumera cada item com números. Ideal para listas de instruções, rankings, ou qualquer lista onde a sequência dos itens é crucial.
  • <li> (List Item): representa um item individual dentro de qualquer lista (<ol>, <ul>).
  • <ul> (Unordered List): representa uma lista não ordenada, geralmente com marcadores.
  • <dl> (Description List): lista de definição usada para agrupar termos e suas descrições.
  • <dt> (Description Term): usado para indicar o termo que está sendo definido.
  • <dd> (Description Details): usado para fornecer a definição ou descrição do termo.
