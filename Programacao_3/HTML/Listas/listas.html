Tipos de listas HTML

Listas ordenadas:

As listas ordenadas usam números para identificar os itens desta lista.

  1. 1º item da lista
  2. 2º item da lista
  3. 3º item da lista
  4. 4º item da lista

Listas não ordenadas:

As listas não ordenadas usam marcadores para identificar os itens.

  • Abacaxi
  • Maçã
  • Pêssego

Listas de definição:

HTML
Linguagem de Marcação de HyperText usada para criar páginas web
CSS
Folhas de Estilo em Cascata, usadas para estilizar a aparência de elementos HTML
JavaScript
Linguagem de programação utilizada para tornar as páginas web interativas
