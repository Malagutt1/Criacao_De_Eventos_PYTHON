Introdução ao HTML e CSS

Conceitos Básicos

O <!DOCTYPE html> não é uma tag propriamente dita, mas sim uma declaração especial. Ela orienta o navegador sobre qual versão do HTML deve ser usada para exibir a página. Usaremos <!DOCTYPE html>, que indica o uso da versão atual, HTML5.

Principais tags HTML

  1. Parágrafo: <p>
  2. Elementos para ênfase: <strong>, <em>, <small>, entre outros
  3. Imagens e legendas: <figure>, <img src="" alt="">, <figcaption>
