HTML

Hypertext Markup Language

HTML utiliza "tags" para delimitar e estruturar diferentes tipos de conteúdo em uma página web. Cada tag é envolvida por sinais de menor e maior (< e >), por exemplo:

Um documento HTML básico possui uma estrutura que divide o conteúdo em cabeça (head) e corpo (body).


Principais TAGS do HTML

  • <h1> </h1> <!-- Cabeçalho principal da página -->
  • <h2> </h2> <!-- Subcabeçalho -->
  • <h3> </h3> <!-- Seção -->
  • <h4> </h4> <!-- Subseção -->
  • <h5> </h5> <!-- Título de menor hierarquia -->
  • <h6> </h6> <!-- Título com menor importância -->
  • <p> </p> <!-- Parágrafo de texto -->
  • <img src="html.png" alt="Imagem HTML"> <!-- Inserção de imagem com texto alternativo -->

Caminho Absoluto:

O caminho absoluto mostra a localização completa de um arquivo ou pasta a partir da raiz do sistema de arquivos.

Em sistemas UNIX, a raiz é representada por "/", enquanto no Windows, utiliza-se "\".


Caminho Relativo:

É uma forma prática de apontar para arquivos considerando a localização atual do arquivo que está sendo editado.

  • ./Documento.txt – arquivo na mesma pasta
  • img src="diretorio/arquivo" – arquivo em uma subpasta
  • img src="../arquivo" – arquivo na pasta pai (um nível acima)

Header

O elemento <header> é usado para identificar o cabeçalho de uma página ou seção, podendo conter títulos, logotipos, menus, entre outros.

Nav

O <nav> agrupa links de navegação, facilitando o acesso às diferentes partes do site.

Figure e Figcaption

O <figure> é utilizado para inserir imagens e elementos visuais, enquanto <figcaption> fornece a legenda explicativa.

Footer

O <footer> indica o rodapé da página ou seção, geralmente contendo informações como direitos autorais e autoria.
