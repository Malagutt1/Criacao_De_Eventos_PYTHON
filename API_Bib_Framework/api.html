O que é uma API?

As APIs são um conjunto de rotinas e métodos que permitem que o desenvolvedor “importe” ou “empreste” informações ou funcionalidades de outras aplicações para o seu projeto ou aplicação.

Ela é usada como um "garçom" entre o cliente e o servidor, enviando e recebendo dados, criando uma integração e interoperação sem a necessidade do sistema consumidor ter detalhes de implementação do software.

Por que isso é bom?
Isso torna o uso do software mais fácil para os usuários, ajuda os desenvolvedores a trabalharem mais rápido, facilita a criação de novas funções e permite que diferentes sistemas e empresas trabalhem juntos de forma mais eficiente.

Um excelente exemplo é a API do Google Maps, que se espalhou por quase todos os sites de empresas dos mais diversos segmentos, potencializando a usabilidade e os serviços em si.
